## 📝 项目简介 (About The Project)

**ALLEN哥Python工具箱 3.0** 是一个基于 Python 开发的多功能命令行界面（CLI）应用程序，它将五个不同场景的实用小工具整合在一个脚本中，让用户可以通过简单的数字选择来执行对应的任务

无论你是需要快速计算全班的英文平均分，还是想为下一次的旅行做预算分配，这个工具箱都能为你提供快速的解答

---

## 🚀 功能矩阵 (Features Matrix)

工具箱内置以下五大核心功能，请在主菜单输入对应数字启动：

| 选项 | 工具名称 | 功能描述 | 状态 |
| :---: | :--- | :--- | :--- |
| **1** | 🏋️‍♂️ **BMI Tools** | 计算用户的身高体重指数 (BMI) 与体脂报告，内置仿真的终端机加载动画 | `[VIP 专属]` |
| **2** | 📚 **ENG AVG Tools** | 英文成绩平均分计算器，支持连续输入多笔成绩，输入 `-1` 即可结算 | `[Free]` |
| **3** | ✈️ **Travel Plan TimeTable** | 旅游计划与预算表，输入天数、人数与总预算，自动生成精美的行程信息卡 | `[Free]` |
| **4** | 🛍️ **Purchase Amount** | 购物折扣计算器，根据会员等级（VIP/一般/非会员）与消费门槛自动计算最终结账金额 | `[Free]` |
| **5** | 🪪 **Personal DATA Card** | 个人信息名片生成器，收集基本资料并在终端机渲染出一张专属的文字名片 | `[Free]` |

## 💻 系统需求与安装 (Installation)

本程序无需安装任何第三方库，只需确保你的电脑已安装 Python 环境

1. **环境要求**: Python 3.x 
2. **下载脚本**: 将 `ALLEN哥Python工具箱 3.0 .py` 下载到本地
3. **运行程序**:
   打开终端机 (Terminal) 或命令提示符 (CMD)，导航至脚本所在目录，并输入：
   ```bash
   python "ALLEN哥Python工具箱 3.0 .py"
